# Results

This folder contains experimental outputs from training and evaluation:

- `figures/` – Accuracy curves, loss plots, confusion matrices
- `metrics/` – CSV summaries of model performance
- `logs/` – Training and inference logs

Key Findings:
- GPT-4o achieved 87% accuracy with only 3 epochs per phase.
- Hybrid ensemble improved performance to 89%.
- CNN baselines required significantly more epochs and compute.
